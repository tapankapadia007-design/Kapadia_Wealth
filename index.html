How to get Access Token daily
1
Click "Open Zerodha Login" → login with your credentials
2
After login you'll be redirected. Copy the request_token from the URL
3
On your PC run this Python command:

python -c "from kiteconnect import KiteConnect; k=KiteConnect(api_key='YOUR_KEY'); s=k.generate_session('REQUEST_TOKEN','YOUR_SECRET'); print(s['access_token'])"
4
Paste the printed token above and click Connect
