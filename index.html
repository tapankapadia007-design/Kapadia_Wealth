⚡ Kite Connect Setup
Real-time NSE volume spike screener using Zerodha WebSocket.
Aggregates all NSE ticks into 30-second (0.5 min) OHLCV candles and flags volume spikes.
📋 Kite Developer App Settings:
Redirect URL  → http://127.0.0.1:5500/  (VS Code Live Server port)
Postback URL → http://127.0.0.1:5500/postback

After registering, grab your API Key and API Secret below.
⚠️ Token Exchange is server-side only.
Run this in terminal after getting the request_token:

pip install kiteconnect
python -c "from kiteconnect import KiteConnect; k=KiteConnect(api_key='YOUR_KEY'); s=k.generate_session('REQUEST_TOKEN','YOUR_SECRET'); print(s['access_token'])"

Then paste the printed access_token into the field above and click "I already have an access token".
How This Screener Works
1
Fetches all NSE EQ instruments (~2000 stocks) from api.kite.trade/instruments/NSE
2
Subscribes all tokens via Kite WebSocket in FULL mode — receives live ticks with LTP, volume, OHLC
3
Aggregates ticks into 30-second OHLCV candles per stock (candles close every 30s)
4
On each candle close: compares current candle volume vs N-candle rolling average — flags 2x/5x/10x spikes
5
Shows real-time alerts, sortable table, and running ticker for all detected spikes
